## 👨‍💻 About

I build **LLM-powered systems end to end**: retrieval-augmented generation (RAG), AI agents,
multimodal pipelines (text · audio · vision · documents), fine-tuning and the MLOps around it.
I trained and published **my own Portuguese LLM**, and I architect full-stack products on top —
from the data layer and vector search to the UI and the cloud/disaster-recovery design.

- 🧠 **AI/ML:** RAG · AI Agents · Fine-tuning (QLoRA) · Embeddings · Multimodal · MLOps
- 🏗️ **Architecture:** system design · vector databases · event pipelines · human-in-the-loop
- 🌎 Based in Brazil · open to remote roles
